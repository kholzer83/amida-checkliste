# amida-checkliste

Checkliste für Vermietungsprozesse – 4 Wiener Mietwohnungen, je 12 Prozessschritte.

## Features
- Dashboard mit Übersicht aller Objekte und nächsten Terminen
- Fortschrittsbalken + Dringlichkeitsanzeige pro Objekt
- Bearbeitbare Notizen, Terminverwaltung, Schritte mit Zeitstempel
- Alle Daten lokal im Browser (localStorage) – keine Server, kein Login
- Installierbar als PWA (Home-Screen-Icon)

## Deployment

Gehostet via GitHub Pages:  
**https://kholzer83.github.io/amida-checkliste**

## iPhone Installation

1. Safari öffnen → https://kholzer83.github.io/amida-checkliste aufrufen
2. Teilen-Button antippen (Rechteck mit Pfeil nach oben)
3. „Zum Home-Bildschirm" wählen
4. App erscheint wie eine native App ohne Browser-Leiste

## Lokale Nutzung

Einfach `index.html` im Browser öffnen – kein Build-Schritt nötig.
